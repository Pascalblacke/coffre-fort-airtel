# Airtel SafeLock (Prototype)

Auteur : BOULOUDI MOUNGUENGUI CLAINE PASCAL  
Pays : Gabon  
Date : 13/04/2026

## Description
Airtel SafeLock est une solution digitale d’épargne verrouillée permettant aux utilisateurs de bloquer une somme d’argent jusqu’à une date définie, sans possibilité de retrait anticipé.

## Fonctionnalités
- Création de coffres d’épargne
- Blocage des fonds jusqu’à échéance
- Suivi des montants verrouillés
- Simulation de cas d’usage (école, voyage, etc.)

## Statut
Prototype fonctionnel (web app)
